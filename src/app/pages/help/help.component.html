Help
Step 1: Register/Login into TaskAssigner Application.

Step 2: After login you will create a company if the company is already created, you will go to the dashboard where you will see all the project and all tasks.

Step 3: In Projects you can create multiple Projects Inhouse and Outer Project. You can upload multiple documents in Projects.

Step 4: In Tasks you can create multiple tasks that you can assign to any user. You can upload multiple documents in Tasks.

Step 5: In the member, you can invite anyone by email and direct add details of any employee for add in the Task Assigner.
